## 🎨 Animation System Deep Dive

### Core Animation Philosophy
Every animation follows these principles:
- **Natural Motion** - Spring physics instead of linear easing
- **Performance** - GPU-accelerated (transform, opacity only)
- **Purpose** - Each animation enhances understanding
- **Subtlety** - Animations support content, don't distract
- **Consistency** - Similar elements behave similarly

### Spring Physics Configuration

```typescript
// Premium spring settings
const springConfig = {
  stiffness: 350,    // How quickly it reaches target
  damping: 25,       // How much it bounces
  mass: 1,           // Weight of the element
};

// Smooth spring for large elements
const smoothSpring = {
  stiffness: 300,
  damping: 30,
  mass: 1.2,
};
```

### Animation Components Library

#### Page Transitions
```tsx
<PageTransition>
  {children}
</PageTransition>
```
- Fades in page content
- Triggered on route change
- Uses AnimatePresence

#### Fade In Up
```tsx
<FadeInUp delay={0.2}>
  <YourContent />
</FadeInUp>
```
- Starts below, fades in while rising
- Configurable delay for staggering
- Viewport detection with IntersectionObserver

#### Slide In Variants
```tsx
<SlideInLeft delay={0.1}>Content</SlideInLeft>
<SlideInRight delay={0.1}>Content</SlideInRight>
<SlideInDown delay={0.1}>Content</SlideInDown>
```
- Slides from specified direction
- Spring easing
- Scroll-triggered

#### Stagger Container
```tsx
<StaggerContainer staggerDelay={0.1}>
  <Child1 />
  <Child2 />
  <Child3 />
</StaggerContainer>
```
- Children animate in sequence
- Configurable delay between items
- Perfect for lists and grids

#### Scale & Zoom
```tsx
<ScaleIn delay={0.2}>
```

- Scale: 0.8 → 1.0

- **HoverCard**: Lifts on hover with shadow

#### Animated Counter

- Counts from 0 to target value

### Scroll-Triggered Animations

All animations activate when element enters viewport:

```typescript
const { ref, inView } = useInView({
  triggerOnce: true,   // Animate only once
  threshold: 0.1,      // 10% of element visible
});

<motion.div
  ref={ref}
  initial={{ opacity: 0, y: 20 }}
  animate={inView ? { opacity: 1, y: 0 } : {}}
  transition={{ duration: 0.6, ease: [0.25, 0.1, 0.25, 1] }}
/>
```

### Advanced Interactions

#### Custom Cursor
- Follows mouse with spring delay
- Scales on hover over links/buttons
- Changes color for different elements
- Glow effect follows cursor
- Desktop only (hidden on touch devices)

#### Magnetic Elements
Buttons attract cursor within radius:
```typescript
const handleMouseMove = (e) => {
  const rect = ref.current.getBoundingClientRect();
  const x = e.clientX - rect.left - rect.width / 2;
  const y = e.clientY - rect.top - rect.height / 2;
  
  const distance = Math.sqrt(x * x + y * y);
  if (distance < 100) {
    x.set(x * 0.3);
    y.set(y * 0.3);
  }
};
```

#### Floating Particles
```tsx
{particles.map((particle, i) => (
  <motion.div
    key={i}
    animate={{
      x: [0, particle.x, 0],
      y: [0, particle.y, 0],
      opacity: [0.2, 0.5, 0.2],
    }}
    transition={{
      duration: particle.duration,
      repeat: Infinity,
      ease: "easeInOut",
      delay: particle.delay,
    }}
  />
))}
```

#### Gradient Orbs
```tsx
<motion.div
  animate={{
    x: [0, 100, 0],
    y: [0, 50, 0],
    scale: [1, 1.2, 1],
    opacity: [0.3, 0.5, 0.3],
  }}
  transition={{
    duration: 20,
    repeat: Infinity,
    ease: "easeInOut",
  }}
  className="absolute w-[600px] h-[600px] bg-gradient-to-br from-blue-400/20 to-cyan-400/20 rounded-full blur-3xl"
/>
```

### Performance Best Practices

✅ **DO:**
- Use `transform` and `opacity` only
- Add `will-change: transform` for animating elements
- Use `layoutId` for shared element transitions
- Implement `useReducedMotion` for accessibility
- Lazy load animation components

❌ **DON'T:**
- Animate `width`, `height`, `top`, `left`
- Animate on every scroll event (use throttle)
- Create too many simultaneous animations
- Forget to clean up event listeners
- Ignore mobile performance

### Accessibility

```tsx
const prefersReducedMotion = useReducedMotion();

<motion.div
  animate={prefersReducedMotion ? {} : { x: 100 }}
/>
```
